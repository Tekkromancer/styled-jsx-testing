# RepairClinic

## Running the App
### Dev Move
To run the app in development mode, run `yarn dev`.
### Production
To build and serve the site, run:
```
> yarn build
> yarn start
```
Running `yarn prod` will run both steps.  Note that the site will start on port 8000 instead of 3000 when in "production" mode.
